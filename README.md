# MoodPlayer
JavaFX Musikplayer mit stimmungsbasierter Filterung. Filtert Songs nach 12 Moods, Live-Visualisierung mit rotierender Vinyl-Animation, animierten Lava-Hintergrung und organischer Mood-Auswahl. Implementiert mit MVC-Architektur, Observer Pattern, Multithreading  für flüssige GUI und Custom Components mit Kollisionserkennung.
